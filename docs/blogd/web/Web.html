    Class Web

    The class which outlines the blog.spaces functionality

    This class is registered with the URLRouter as a web interface. It's scope is to allow users to create and read blogs, and for users to interact with their accounts. The following route conventions will be used:

    • "/": home, login/logout

    • "/user/": create account, view/update account details

    • "/read/": reading/searching existing blog posts

    • "/write/": writing blog posts
